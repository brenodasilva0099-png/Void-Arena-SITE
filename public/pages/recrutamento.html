Recrutamento / Peneira

Solicitações de clube

Capitães podem chamar jogadores. Jogadores sem clube podem pedir avaliação para entrar em um time.

?

Enviar solicitação

Para recrutar, primeiro escolha um jogador no Banco de Jogadores. Para peneira, selecione o time desejado aqui.

Ver jogadores
Para recrutamento, selecione um jogador na tela Jogadores. Para peneira, selecione um time.
Como funciona

O capitão só consegue enviar recrutamento pelos times que ele gerencia. O jogador pode pedir peneira para qualquer time cadastrado.

Histórico de solicitações

Tudo que for enviado fica salvo no banco atual para a staff acompanhar depois.
