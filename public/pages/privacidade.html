Void Arena Legal

Política de Privacidade

Como a Void Arena coleta, usa, armazena e protege dados necessários para o funcionamento do site e do bot.

?

Última atualização: 05/07/2026

1. Dados que podemos coletar

Para operar a Void Arena, podemos coletar e armazenar dados necessários ao funcionamento do site, bot, eventos, permissões, rankings e integrações.

👤 Dados de login e sessão. 💬 Discord ID, nome, avatar e cargos quando usados para login/permissão. 🛡️ Dados de times, capitães, titulares, reservas, tags e logos. 🏆 Inscrições, eventos, validações, resultados, rankings e pontuações. 📎 Mensagens, prints, anexos e comprovantes enviados para validação. 🔗 Conexões informadas pelo usuário, como Steam, Xbox, TikTok, Spotify, EA, Riot, PSN e outras.

2. Como usamos esses dados

Os dados são usados para autenticar usuários, controlar permissões, exibir perfis públicos, organizar times, registrar eventos, gerar chaveamentos, validar resultados, manter rankings, enviar notificações, criar filas, calls e painéis do bot.

Também podemos usar dados técnicos para corrigir bugs, evitar perda de informações, melhorar segurança e manter backups do projeto.

3. Perfis públicos e informações visíveis

Algumas informações podem ficar visíveis para outros usuários dentro da Void Arena, como nome público, avatar, time, tag, posição, região, perfil de time, ranking, estatísticas e conexões que você preencher publicamente.

Evite cadastrar dados sensíveis, privados ou que você não queira exibir para outros membros da comunidade.

4. Integração com Discord e bot

Quando você usa login com Discord ou funções do bot, a Void Arena pode utilizar seu Discord ID, avatar, nome público, cargos do servidor e mensagens necessárias para validar permissões, inscrições, filas, calls, resultados e notificações.

O bot pode criar ou atualizar mensagens, canais, calls privadas, painéis, validações e registros necessários para o funcionamento dos eventos.

5. Compartilhamento de dados

A Void Arena não vende dados dos usuários. Os dados podem ser exibidos internamente no site, no bot ou para a staff quando forem necessários para administração, suporte, validação, segurança, histórico ou resolução de problemas.

Dados também podem passar por serviços técnicos usados para hospedar e operar o projeto, como plataforma de hospedagem, GitHub/backup e Discord, conforme necessário para funcionamento.

6. Armazenamento e backups

As informações podem ser armazenadas no banco atual do projeto e em backups operacionais para evitar perda de dados de times, eventos, rankings e validações.

Backups são usados para recuperação técnica e administração do projeto, não para venda ou exposição pública de dados.

7. Remoção, correção e exclusão

Você pode solicitar correção ou remoção de dados à staff da Hollow Nexus. Contas locais criadas fora do Discord podem ser excluídas pelo perfil quando a opção estiver disponível.

Alguns registros podem precisar ser mantidos por segurança, histórico de evento, comprovação de validação, prevenção de fraude ou integridade competitiva.

8. Segurança

A Void Arena aplica controles de sessão, permissões por cargo, rotas protegidas e separação de áreas administrativas. Ainda assim, nenhum sistema online é totalmente imune a falhas.

Não compartilhe senha, token, dados sensíveis ou informações privadas nos campos públicos do site ou em canais abertos.

9. Plataformas externas

A Void Arena pode conter links ou referências a Discord, Steam, EA, Xbox, TikTok, Spotify, YouTube, Twitch, Riot, PlayStation Network e outras plataformas. Cada serviço externo possui suas próprias regras e políticas.

A Void Arena / Hollow Nexus não é afiliada, patrocinada, endossada ou administrada por essas plataformas.

10. Contato

Para dúvidas, revisão, remoção, correção de dados ou problemas de privacidade, procure a staff responsável pelo servidor Hollow Nexus / Void Arena.
